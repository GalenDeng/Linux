# mac install software operation (2019.07.02)

1. `具体操作`
```
1、打开终端，习惯性命令：
brew update
//结果：Already up-to-date.

2、终端继续执行命令：
brew search nginx   //查询要安装的软件是否存在

3、这里我们多执行一步“废”命令，不过有利于我们后面的配置：
brew info nginx

4. 安装
brew install xxx
```
