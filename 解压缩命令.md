## 解压缩命令 （2019.02.22）
1. `tar.xz` : `目前压缩率最好的格式`
*  解压和创建 XZ 文件
```
解压 ： tar -Jxf linux-3.12.tar.xz
创建 ： tar -Jcf linux-3.12.tar.xz linux-3.12/
```
*  `tar.bz2`: `比 tar.xz 次一点的压缩率的格式`
```
解压 tar -xjf all.tar.bz2
创建 tar -cjf all.tar.bz2 *.jpg
```
* `rar文件解压缩`
```
* 解压：unrar x 课件.rar
* 压缩文件A和B：rar a 压缩后.rar A B
```
